# My blog

一个使用**Vue3**+**Unocss**搭建的个人博客。

本博客是一个究极缝合怪，也是我第一个完全从0开发的项目。

- 背景参考 [小孙同学的个人主页](https://github.com/sun0225SUN/star-trail) 
- 界面参考 [Moeyy 的博客](https://moeyy.cn/)



## 特点

- 采用**Unocss**原子引擎
- 使用**阿里巴巴图标库**
- 支持**highlight.js**代码高亮
- 支持**katex**数学公式
- 支持**fancybox**图片灯箱
- 支持**响应式**布局
- **丝滑的动画**和**简洁的界面**
- 更多功能开发中......
